# Intellij IDEA快捷键 - OSX

## Mac键盘标志对应表

| 标志 | 按键说明 |
| :--: | :------: |
|  ⌘   | Command  |
|  ⇧   |  Shift   |
|  ⌥   |  Option  |
|  ⌃   | Control  |
|  ↩︎   |  Return  |
|  ⌫   |  Delete  |
|      |          |

## 常用快捷键

- `⌘+C` 复制当前行或选定的块到剪贴板
- `⌘+D` 复制(Duplicate)当前行或选定的块
- `⌘+X` 剪切当前行或选定的块到剪贴板
- `⌘+V` 从剪贴板粘贴
- `⌘+Z` 撤销
- `⌘+⌫` 删除当前行或选定的块的行
- `⌘+⌥+T` 包围选中的代码
- `⌘+W` 关闭活动的编辑器选项卡
- `⌘+F` 文件内查找
- `⌘+R` 文件内替换
- `⌘+⇧+F` 全局查找
- `⌘+L` 跳转到指定行列
- `⌘+E` 显示最近打开文件记录列表
- `⌘+B` 进入光标所在的方法/变量的接口或是定义处
- `⌘+Left/Right` 移动光标到前/后单词
- `⌘+⇧+Left/Right` 选中光标前后代码
- `⌥+Up/Down` 连续选中代码块/减少当前选中的代码块
- ``⌘+P` 显示方法参数信息
- `⌃+J` 快速查看文档
- `⌃+Up/Down`跳转到上一个/下一个方法
- `⌃+Left/Right` 跳转到左一个/右一个标签页
